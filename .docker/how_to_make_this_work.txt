QUICK HOW TO START THIS

(Windows)
1. Sciagnij i zainstaluj dockera - https://docs.docker.com/docker-for-windows/install/
2. Mozliwe ze jakies resety kompa beda potrzebne i zezwolenie na uzywanie czegośtam
3. Dodaj dysk C:/ do shared drives - 
    3.1) Prawy dolny rog - ikonka dockera - Settings
    3.2) Shared drives
    3.3) Zaznacz dyck C i kliknij Apply
    3.4) Pewnie bedzie trzeba poczekac chwile na reset
    3.5) Done
4. Odpal Powershell w folderze z docker-compose (mozliwe ze cmd tez zadziala ale nie chce mi sie sprawdzac)
5. Wpisz komende - 'docker-compose up -d' (troche sie rzeczy bedzie pobierac za pierwszym razem)
6. Sprawdz dla pewnosci czy kontenery wstaly - 'docker container ls -a'
7. Jesli kontenery stoja to jest ok (STATUS: Up)

8. Mozna działać z baza przez pgadmin (jak nie chcesz to mozna wywalic z docker-compose i uzywac jakiegos dbeavera)
9. pgadmin bedzie stał na http://localhost:5050/ z passami - email: sa i password - sa
10. Zeby dodac tam serwer trzeba w zakladce Connection jako host dac 'postgres' i username: sa, password: sa, reszte zostawic jak jest

11. Jak chcesz korzystac z innej apki to dodaj serwer na http://localhost:5432 i takimi samymi passami

JAK WYŁĄCZYĆ?
12. Ikonka dockera -> Quit docker desktop lub komendami -> 'docker kill $(docker container ls -a -q)' i 'docker rm $(docker container ls -a -q)' <- opcjonalne jak chcesz zrobic wipe
13. Jak komendami to sprawdz jeszcze czy kontenery padly - 'docker container ls -a'
